# Integration with External Systems

Use SAP Advanced Financial Closing to trigger jobs in an existing external system.



<a name="loio90573aec75b1402897260347e95d94c3__section_sl2_kwf_cfc"/>

## Key Features



-   Implementation of the scheduling provider interface based on an intuitive OpenAPI specification
-   Perform the retrieval of job definitions as well as the scheduling and synchronization of jobs for external systems
-   Deal with external jobs and built-in task types within the same task list.



<a name="loio90573aec75b1402897260347e95d94c3__section_q3n_kwf_cfc"/>

## What External System Integrations are Possible?

SAP Advanced Financial Closing can integrate with any external system that implements a simple scheduling provider interface.



<a name="loio90573aec75b1402897260347e95d94c3__section_ll1_lwf_cfc"/>

## Integration with SAP Advanced Financial Closing

You can set up an integration between your external system and SAP Advanced Financial Closing by implementing a simple scheduling provider interface. Once you've performed the required setup steps, SAP Advanced Financial Closing and the external system exchange information in a standardized way through this scheduling provider interface. This allows the scheduling of custom jobs in external systems in addition and in parallel to the built-in task types that SAP Advanced Financial Closing supports out of the box, thus allowing you to integrate closing steps performed in the external system into your task lists in SAP Advanced Financial Closing.
