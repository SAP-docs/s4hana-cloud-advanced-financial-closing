# Integration with SAP Accounting Automation by BlackLine for SAP Advanced Financial Closing

Use SAP Advanced Financial Closing to trigger processes in your existing BlackLine system.



<a name="loiobe3f4883cf914ff7810f531dde43ef17__section_sl2_kwf_cfc"/>

## Key Features



-   Integration of BlackLine processes into the financial close in SAP Advanced Financial Closing
-   Automatic triggering of BlackLine processes based on scheduling capabilities of SAP Advanced Financial Closing



<a name="loiobe3f4883cf914ff7810f531dde43ef17__section_q3n_kwf_cfc"/>

## What is SAP Accounting Automation by BlackLine?

BlackLine is a software company that provides solutions for financial close and accounting processes. With SAP Accounting Automation by BlackLine, SAP and BlackLine are partnering up to allow for a seamless financial close process between the two systems. ## Integration with SAP Advanced Financial Closing

You can set up an integration between SAP Accounting Automation by BlackLine and SAP Advanced Financial Closing. Once you've performed the required setup steps, you can use SAP Advanced Financial Closing to trigger processes that you've defined in your BlackLine system. This allows you to integrate closing steps performed with SAP Accounting Automation by BlackLine into your closing task lists in SAP Advanced Financial Closing.
