# How to Add the Launchpad Tile in SAP S/4HANA Cloud Public Edition

Add a tile for SAP Advanced Financial Closing to your launchpad in SAP S/4HANA Cloud Public Edition.



<a name="copy857efd551e6341088b5e4da6e38c1ea0__context_wfp_bsl_ngc"/>

## Context

You can grant users the authorization to see the *Manage Advanced Financial Closing* tile on the launchpad of their SAP S/4HANA Cloud Public Edition system to navigate directly to SAP Advanced Financial Closing on the SAP BTP.



## Procedure

For more information about how to authorize users to see the *Manage Advanced Financial Closing* tile on their launchpad in SAP S/4HANA Cloud Public Edition, see [How to Maintain Business Roles](https://help.sap.com/viewer/7a732dca39e2412cb8661b769277bcbb/SHIP/en-US/eb0f7719da404f1ca48f28a4cba06f35.html "Use business catalogs and IAM apps to grant authorizations.") :arrow_upper_right:.
